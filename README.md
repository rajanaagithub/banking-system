Overview.....
This is a console-based Banking System developed using Java and Object-Oriented Programming (OOP) principles.
The application allows users to perform basic banking operations like account creation, deposit, withdrawal, and balance checking.
Features
 Create a bank account
 Deposit money
 Withdraw money
 Check account balance
 Display account details
 Input validation (invalid amount, insufficient balance)
 Tech Stack
Language: Java
Concepts Used: OOP (Encapsulation, Classes, Methods)
Tools: VS Code / IntelliJ IDEA
Project Structure
BankingSystem/
│── BankApp.java
▶️ How to Run
Step 1: Clone the repository
git clone https://github.com/your-username/BankingSystem.git
Step 2: Navigate to the project folder
cd BankingSystem
Step 3: Compile the program
javac BankApp.java
Step 4: Run the program
java BankApp
